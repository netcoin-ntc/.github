## About Netcoin

Netcoin implements Proof of Network (PoNW), a revolutionary consensus mechanism that makes computational energy productive by embedding proof-of-work puzzles directly into network routing decisions.

## Getting Started

1. **Run a Node**: `git clone https://github.com/netcoin-ntc/netcoin && cd netcoin && make deploy`
2. **Create Wallet**: `netcoin-node wallet create`
3. **Earn NTC**: Nodes automatically mine through routing work
4. **Premium Services**: Use earned NTC for QoS routing, VPN, etc.
